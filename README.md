# Data Sources

A curated collection of datasets designed for Data Science, Machine Learning, Statistical Analysis, and EDA (Exploratory Data Analysis) projects.

## Purpose

This repository serves as a centralized catalog of publicly shareable datasets used for:

* Exploratory Data Analysis (EDA)
* Data Cleaning and Preprocessing
* Feature Engineering
* Statistical Modeling
* Machine Learning Training
* Regression Problems
* Classification Problems
* Data Visualization
* Educational and Research Projects

## Repository Structure

Each dataset contains:

* Raw source files
* Dataset-specific documentation
* Suggested analytical use cases
* Metadata describing attributes and targets when available

## Current Datasets

### Car Price Dataset

Vehicle specifications and market pricing information suitable for:

* EDA
* Price prediction
* Regression modeling
* Feature importance analysis
* Automotive market research

### Bipolar Symptoms Dataset

Mental health symptom observations and diagnostic labels suitable for:

* EDA
* Classification modeling
* Feature selection studies
* Healthcare analytics research
* Predictive diagnosis experiments

## Objectives

The goal of this repository is to provide clean, well-organized datasets that can be immediately used in:

* Jupyter Notebooks
* Python Data Science workflows
* Machine Learning pipelines
* Academic assignments
* Portfolio projects
* Benchmark experiments

Contributions of additional datasets, documentation improvements, and metadata enhancements are welcome.
